# Installationsanweisungen
Um die erforderlichen Abhängigkeiten zu installieren, führen Sie bitte den folgenden Befehl in Ihrem Terminal aus:

`pdm install`

Durch Ausführen dieses Befehls werden alle notwendigen Pakete und Bibliotheken installiert, die für das Projekt benötigt werden.
