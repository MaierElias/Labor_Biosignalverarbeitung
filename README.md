## Das ist die README-Datei für das Biosignalverarbeitungs-Laborprojekt
# Installationsanweisungen
Um die erforderlichen Abhängigkeiten zu installieren, muss folgender Befehl im Terminal ausgeführt werden:

`pdm install`

Durch Ausführen dieses Befehls werden alle notwendigen Pakete und Bibliotheken installiert, die für das Projekt benötigt werden.

Damit die LaTeX-Dokumente korrekt kompiliert werden können, wird empfohlen folgende LaTeX-Pakete zu installieren:
- LaTeX Workshop (für VSCode)
- MikTeX (für Windows unter folgendem Link: https://miktex.org/download)
- Strawberry Perl (für Windows unter folgendem Link: https://strawberryperl.com/, hier kann die .msi Datei heruntergeladen und installiert werden)

# Projektübersicht
Im Unterordner `Labor_1` befindet sich der Code für das erste Laborprojekt und
im Unterordner `Labor_2` der Code für das zweite Laborprojekt.

Im Unterordner `Bericht_Biosignalverarbeitung` befindet sich der Bericht zum Laborprojekt 1 und im Unterordner `Bericht2_Biosignalverarbeitung` der Bericht zum Laborprojekt 2.
